# 🌸 Portfólio — Mayara Silva

Meu portfólio pessoal, desenvolvido do zero com HTML5, CSS3 e JavaScript puros, sem frameworks. Um espaço para reunir minha trajetória, formação e projetos enquanto estudo Análise e Desenvolvimento de Sistemas.

🔗 [https://maysilvalps.github.io/portfolio-mayara-uninter/](https://maysilvalps.github.io/portfolio-mayara-uninter/)

---

## ✨ Funcionalidades

- **Tema claro/escuro** — alternância dinâmica de cores com um clique
- **Layout responsivo** — adaptado para desktop, tablet e mobile
- **Formulário de contato validado** — verificação de campos obrigatórios e formato de e-mail, sem recarregar a página
- **Mascote interativo** — os "olhos" seguem o cursor do mouse e a "boca" reage ao preenchimento do formulário
- **Revelação suave das seções** ao rolar a página

## 🛠️ Tecnologias

- HTML5
- CSS3 (variáveis CSS para o sistema de temas, Flexbox, media queries)
- JavaScript (Vanilla JS — manipulação de DOM, eventos e `IntersectionObserver`)

```bash

cd portfolio-mayara-uninter
```

Depois é só abrir o `index.html` no navegador (ou usar a extensão *Live Server* do VS Code).
