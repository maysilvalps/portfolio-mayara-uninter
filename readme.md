# Portfólio Pessoal - Mayara Silva

Portfólio desenvolvido para apresentar minha trajetória profissional, formação e projetos na área de tecnologia.

## Sobre o Projeto

Este site foi desenvolvido com foco em criar uma experiência moderna e acessível, utilizando conceitos de front-end e um design limpo e delicado.

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

## Seções do Site

* **Sobre mim:** Apresentação pessoal, formação acadêmica na Uninter (Polo Rondonópolis) e objetivos profissionais.
* **Tecnologias:** Ferramentas e linguagens dominadas e em estudo.
* **Projetos:** Lista de aplicações práticas desenvolvidas.
* **Contato:** Canais de comunicação direta (GitHub, LinkedIn, Instagram e E-mail).

## Acesso

Você pode acessar o projeto online através do link do repositório ou visualizar a página hospedada.
